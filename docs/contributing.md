# Contributing

## Core Contribution Rules

- Preserve neutral, evidence-based language.
- Do not add unsupported accusations.
- Prefer smaller, reviewable ETL steps.
- Add source references for every new live dataset.
- Document every new signal factor before shipping it.
- Prefer insufficient evidence to score over weak joins or undocumented inference.
- Keep demo data synthetic unless explicitly replacing it with real sourced data.

## Good First Contributions

- add a new official federal dataset
- improve parsing resilience
- tighten UI explanations
- improve keyboard and screen-reader accessibility
- add missing data freshness indicators
